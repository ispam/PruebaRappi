
# Prueba Rappi
Prueba de conocimiento para entrevista de trabajo. Deadline 3 diás a partir de Julio 30.

## Arquitectura

1. Las capas de la aplicación (por ejemplo capa de persistencia, vistas, red, negocio, etc) y qué clases pertenecen a cual.
2. La responsabilidad de cada clase creada.

## Preguntas

1. En qué consiste el principio de responsabilidad única? Cuál es su propósito?
2. Qué características tiene, según su opinión, un “buen” código o código limpio?
